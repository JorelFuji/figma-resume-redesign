Figma Case Study

Executive Resume Redesign

A personal redesign project focused on creating a modern, professional resume layout with stronger hierarchy, clearer typography, and a more polished visual presentation.

View Primary Figma Site View Secondary Figma Site

Project Intent

Designed as a portfolio-ready resume system

This project reframes a resume as both a professional document and a design artifact. The focus is on improving scan-ability, composition, and consistency while keeping the final result useful for real-world applications.

Tools

  • Figma
  • PDF export workflow
  • Grid and hierarchy planning
  • GitHub Pages-ready presentation

Goals

Improve readability

Make the resume easier to scan with clearer section rhythm and cleaner spacing.

Strengthen hierarchy

Use typography, sizing, and placement to guide attention toward the most important details.

Keep it reusable

Build a structure that can be updated for future applications without redesigning from scratch.
