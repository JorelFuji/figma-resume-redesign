# Executive Resume Redesign in Figma

A personal design project focused on creating a modern, professional resume layout using Figma.

## Goals

- Improve readability and visual hierarchy
- Build a polished layout suitable for professional roles
- Create reusable design components for future resume versions

## Tools

- Figma
- PDF export
- Typography and grid-based layout principles
