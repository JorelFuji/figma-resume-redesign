# Executive Resume Redesign in Figma

A personal design project focused on redesigning a resume into a cleaner, more modern, and more professional presentation using Figma.

This repository is the showcase layer for the project: it documents the goals behind the redesign, stores exported presentation assets, and provides a lightweight portfolio page for sharing the work online.

## What This Project Is

The goal of this redesign is to turn a standard resume into a stronger visual communication piece without losing clarity. The layout is meant to feel polished and intentional while still staying practical for professional use.

## Design Goals

- Improve readability and visual hierarchy
- Create a stronger first impression through layout and typography
- Build a reusable structure for future resume updates
- Keep the final design professional, clean, and easy to scan

## Tools Used

- Figma
- PDF export
- Grid-based layout planning
- Typography and hierarchy principles
- GitHub for documentation and project presentation
